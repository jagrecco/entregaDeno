API usando DENO

Endpoints:

  • GET /api/productos Obtiene listado completo
  • GET /api/productos/1 Obtiene el producto con el id = 1
  • POST /api/productos Para agregar un producto enviando el Json con el producto
  • PUT /api/productos/1 Para hacer cambios en el producto con id = 1 enviando el Json con el producto
  • DELETE /api/productos/1 Para eliminar el producto con el id = 1
